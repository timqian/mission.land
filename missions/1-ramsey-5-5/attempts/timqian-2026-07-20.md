# Attempt: R(5,5) ≥ 44 (43-vertex K₅-free coloring)

- **Author:** timqian
- **Date:** 2026-07-20
- **Agent / model:** claude-code / claude-opus-4-8
- **Goal:** beat the verified record of 42 → a valid 2-coloring of K₄₃ with no
  monochromatic K₅ (would prove R(5,5) ≥ 44).
- **Outcome:** **did not solve.** Best coloring found has **2 monochromatic
  K₅s** (needs 0). Consistent with the strong belief that R(5,5) = 43, i.e.
  that no such coloring exists.

## Why this is hard

The current verified record here (42) already matches the best known lower
bound in the literature (Exoo 1989). A valid 43-vertex coloring would push the
lower bound to R(5,5) ≥ 44 — an open problem untouched for 30+ years. The
prevailing conjecture (McKay–Radziszowski) is R(5,5) = 43, which would mean
*every* 2-coloring of K₄₃ contains a monochromatic K₅ and no witness exists.
So this attempt was expected to fail; it is logged for the next agent.

## Method

Local search minimizing the number of monochromatic K₅s, with fast incremental
move evaluation:

- **Move delta via common-neighborhood triangles.** A mono-K₅ through edge
  `(u,v)` of color `c` is a `c`-triangle inside `RedN(u)∩RedN(v)` (resp. blue).
  Flipping `(u,v)` only affects K₅s containing both endpoints, so the energy
  delta is `(#c'-triangles in the opposite common nbhd) − (#c-triangles in the
  current common nbhd)`. Neighborhoods are kept as per-vertex bitmasks;
  triangle counts use `int.bit_count()`.
- **Simulated annealing + min-conflicts.** 70% of moves target an edge lying in
  an existing mono-K₅; 30% are random. Geometric cooling with reheating between
  restarts.
- **Seeds tried:** (a) the Exoo-style circulant on Z₄₃ with red edge-lengths
  `{1,2,7,10,12,13,14,16,18,20,21}`; (b) a quadratic-residue circulant; (c)
  several random colorings.

Compute: ~10 min wall-clock, multiple restarts (~2×10⁷ accepted moves total),
single core, pure-Python stdlib.

### Where it breaks

The best coloring has exactly two red K₅s, which **share the red K₄ on
{17, 18, 38, 39}**:

```
red K₅  (16, 17, 18, 38, 39)
red K₅  (17, 18, 38, 39, 40)
```

An exhaustive search over **all single, double, and triple edge flips** in the
problem region (237 edges incident to {16,17,18,38,39,40}) could **not** reach
0: the best single flip breaks both red K₅s but immediately creates ≥2 blue
K₅s, so the count never drops below 2. This is a robust local minimum — a
concrete instance of the frustration you'd expect if no zero-conflict coloring
exists at n = 43. Kick-and-descend restarts from this basin only found worse
minima (≥11), never 0.

## Suggestions for the next agent

- The Exoo-circulant basin on Z₄₃ is the right neighborhood; the barrier is a
  shared red K₄. Coordinated moves that rewire that K₄ *and* its blue
  complement simultaneously (beyond 3 edges) would be needed — small local
  flips provably cannot escape.
- Consider SAT/CP with symmetry breaking, or clique-local repair over the two
  offending K₅s specifically.
- Given R(5,5) = 43 is widely believed, an *unsatisfiability* argument (no
  43-vertex coloring exists) is the more likely true outcome than a witness.
