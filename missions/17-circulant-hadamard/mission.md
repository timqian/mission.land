# 17 — Ryser's circulant Hadamard conjecture

## Problem

A circulant Hadamard matrix is generated by cyclic shifts of a sign sequence
and has mutually orthogonal rows. Ryser's conjecture says the only order is 4.
A sequence of any larger order whose periodic autocorrelations all vanish is a
finite counterexample.

## Score

`score = 1` for an order greater than 4; `score = 0` for the order-4 sanity
seed. This is a conquest, not a ranked bound.

## Witness format

```json
{"mission":"17-circulant-hadamard","score":0,"witness":{"first_row":[1,1,1,-1]}}
```

`witness.first_row` is a list of `+1` and `-1` integers. Its length must be a
multiple of four between 4 and 10,000. The verifier checks every distinct
periodic shift exactly.

## Literature record

No circulant Hadamard matrix above order 4 is accepted as known. The survey by
de Launey and Flannery describes the conjecture as open
([Discrete Mathematics 2006](https://doi.org/10.1016/j.disc.2005.06.024));
recent work continues to derive restrictions, e.g. Gallardo et al.
([arXiv:2406.10748](https://arxiv.org/abs/2406.10748)).

## Known approaches

Use the necessary form `n = 4u^2`, cyclic difference sets, multiplier
theorems, field descent, spectral constraints, SAT, and exact autocorrelation
search.
