# 3 — 램지 수 R(5,5): 하한 끌어올리기

## 문제

R(5,5)는 완전 그래프 K_n의 간선을 어떤 방식으로 2-채색하더라도 반드시 단색 K_5가 생기는 가장 작은 n입니다. 램지 이론이 100년 넘게 발전해 왔지만, 지금까지 알려진 것은 **43 ≤ R(5,5) ≤ 46**이 전부입니다(하한은 Exoo 1989, 상한은 Angeltveit & McKay 2024, [arXiv:2409.15709](https://arxiv.org/abs/2409.15709)).

단색 K_5가 없는 K_n의 간선 2-채색을 하나 제시하면 그것이 곧 R(5,5) > n의 증명입니다. 43개 이상의 정점에서 유효한 채색을 찾아낸다면 헤드라인을 장식할 만한 수학적 성과가 됩니다.

## 점수

`score = n`, 즉 정점의 개수입니다. n개 정점에서 유효한 채색은 R(5,5) ≥ n + 1을 증명합니다.

## Witness 형식

```json
{
  "mission": "3-ramsey-5-5",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 30,
  "witness": {
    "rows": ["01101...", "1011...", ...]
  }
}
```

- `rows[i]`는 길이 n − 1 − i의 문자열로, 간선 (i, i+1), (i, i+2), …, (i, n−1)의 색을 차례로 나열한 것입니다. 즉 인접 행렬의 상삼각 부분이며, 정점 번호는 0부터 시작하고 `'0'`은 빨강, `'1'`은 파랑을 뜻합니다.
- `rows`는 n − 1개의 항목으로 이루어집니다(정점 n−1에는 해당 행이 없습니다).
- 어떤 5개 정점을 골라도 그 사이의 간선 10개가 모두 같은 색이어서는 안 됩니다.

검증: `python3 verify.py <witness.json>`

## 문헌 기록

R(5,5) ≥ 43은 Exoo의 42-정점 채색(1989)에서 나온 결과입니다. 재구성 방법은 *Study of Exoo's Lower Bound for Ramsey number R(5,5)*([arXiv:2212.12630](https://arxiv.org/abs/2212.12630))를 참고하십시오. 이 witness는 아직 `records/`에 수록되어 있지 않으므로, 유효한 42-정점 채색을 제출하면 그대로 검증된 기록 보유자가 됩니다.

## 알려진 접근법

- 간선 채색에 대한 국소 탐색(단색 K_5의 개수를 목적 함수로 삼는 담금질 기법이나 타부 탐색)으로는 정점 수 30대 중반까지 도달할 수 있습니다.
- 가장 좋은 구성은 대수적인 것입니다. Paley 계열 그래프나 블록 순환 채색에서 출발해 국소적으로 보수합니다. Exoo의 42-정점 witness 역시 순환 구성에 기반합니다.
- 한 수를 빠르게 평가하는 것이 관건입니다. 공통 이웃 비트마스크 기법으로 각 간선이 포함된 단색 K_5의 개수를 유지하십시오.
