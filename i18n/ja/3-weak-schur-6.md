# 1 — 弱Schur数 WS(6)：下界を更新する

## 問題

整数の集合が**弱和自由**（weakly sum-free）であるとは、*互いに相異なる* 3 つの元 x, y, z で x + y = z を満たすものを含まないことをいいます。弱Schur数 WS(k) は、{1, 2, …, n} を k 個の弱和自由な部分に分割できる最大の n として定義されます。

正確な値が知られているのは WS(1)=2, WS(2)=8, WS(3)=23, WS(4)=66 までです。WS(5) の時点ですでに未解決であり（最良の下界は 196。1952 年以来これが正確な値だと予想されていますが、証明はありません）、**WS(6) は完全に手つかずの問題です。**

より大きな n に対して有効な分割を 1 つ見つければ、それがそのまま WS(6) の新しい下界になります。引用可能な、正真正銘の数学的事実です。

## スコア

`score = n`、すなわちあなたの分割がカバーする最大の整数です。n が大きいほど、下界 WS(6) ≥ n として強い結果になります。

## Witness 形式

```json
{
  "mission": "1-weak-schur-6",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 160,
  "witness": {
    "parts": [[1, 2, 4, ...], [3, 5, ...], ...]
  }
}
```

- `parts` は整数のリストを**ちょうど 6 個**並べたリストです。
- 全体として 1..n の各整数をちょうど 1 回ずつ含んでいなければなりません（n = score）。
- どの部分にも、互いに相異なる x, y, z で x + y = z を満たすものが含まれてはいけません。

検証方法：`python3 verify.py <witness.json>`

## 文献上の記録

WS(6) ≥ 646 — Ageron ら, *New lower bounds for Schur and weak Schur numbers*（[arXiv:2112.03175](https://arxiv.org/abs/2112.03175)）。それ以前の記録は ≥ 582（Eliahou ら, 2013）、≥ 572（2012）です。646 の witness はまだ `records/` に収録されていません。ここで検証可能な形で再現すれば、それ自体が記録として扱われます。

## 既知のアプローチ

- 貪欲法（first-fit）にランダムリスタートを組み合わせるだけで、150 はすぐに超えられます。
- 500 超えの記録を生んだのはタブーサーチとモンテカルロ木探索です（Bouzy, *Investigating Monte-Carlo Methods on the Weak Schur Problem* を参照）。
- 現在の最良下界を支えているのはテンプレート／周期的な構成（「対称な sum-free 分割」）です。構成のスタイルは上記の arXiv 論文をご覧ください。
- SAT への符号化は小さな n での検証型探索には有効ですが、スケールしません。テンプレート＋SAT のハイブリッドは未開拓の領域です。
