# 3 — ラムゼー数 R(5,5)：下界を押し上げる

## 問題

R(5,5) とは、完全グラフ K_n の辺をどのように 2 色で塗り分けても必ず単色の K_5 が現れる最小の n のことです。ラムゼー理論には 100 年を超える歴史がありますが、現時点で分かっているのは **43 ≤ R(5,5) ≤ 46** ということだけです（下界は Exoo 1989、上界は Angeltveit & McKay 2024、[arXiv:2409.15709](https://arxiv.org/abs/2409.15709)）。

単色の K_5 を含まない K_n の辺 2-彩色をひとつ示せば、それがそのまま R(5,5) > n の証明になります。43 頂点以上で有効な彩色が見つかれば、大きな見出しを飾るほどの数学的成果です。

## スコア

`score = n`（頂点数）。n 頂点で有効な彩色が得られれば、R(5,5) ≥ n + 1 が証明されます。

## Witness フォーマット

```json
{
  "mission": "3-ramsey-5-5",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 30,
  "witness": {
    "rows": ["01101...", "1011...", ...]
  }
}
```

- `rows[i]` は長さ n − 1 − i の文字列で、辺 (i, i+1), (i, i+2), …, (i, n−1) の色を並べたもの、すなわち隣接行列の上三角部分です。頂点番号は 0 始まりで、`'0'` が赤、`'1'` が青を表します。
- `rows` の要素数は n − 1 です（頂点 n−1 の行はありません）。
- どの 5 頂点を選んでも、その間の 10 辺がすべて同色になってはいけません。

検証：`python3 verify.py <witness.json>`

## 文献記録

R(5,5) ≥ 43 は、Exoo による 42 頂点の彩色（1989）によるものです。再構成については *Study of Exoo's Lower Bound for Ramsey number R(5,5)*（[arXiv:2212.12630](https://arxiv.org/abs/2212.12630)）を参照してください。この witness はまだ `records/` に収録されていません。有効な 42 頂点彩色を提出すれば、そのまま検証済み記録の保持者になれます。

## 既知の手法

- 辺彩色に対する局所探索（単色 K_5 の個数を目的関数とする焼きなまし法やタブーサーチ）では、30 台半ばの頂点数まで到達できます。
- 最良の構成は代数的なものです。Paley 型グラフやブロック巡回彩色から出発し、局所的な修復を加えます。Exoo の 42 頂点 witness も巡回構成に基づいています。
- 一手の評価を高速化することが重要です。共通近傍のビットマスクを使うテクニックで、各辺が属する単色 K_5 の個数を常に保持しておきましょう。
