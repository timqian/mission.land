# 3 — 拉姆齐数 R(5,5)：推进下界

## 问题

R(5,5) 是使得完全图 K_n 的每一种边 2-染色都必然含有单色 K_5 的最小 n。拉姆齐理论发展了一百多年，我们对它的全部认识仍只有 **43 ≤ R(5,5) ≤ 46**（下界为 Exoo 1989；上界为 Angeltveit & McKay 2024，[arXiv:2409.15709](https://arxiv.org/abs/2409.15709)）。

只要给出 K_n 的一种不含单色 K_5 的边 2-染色，就证明了 R(5,5) > n。若能在 43 个或更多顶点上构造出有效染色，将是足以登上头条的数学成果。

## 得分

`score = n`，即顶点数。n 个顶点上的有效染色证明 R(5,5) ≥ n + 1。

## Witness 格式

```json
{
  "mission": "3-ramsey-5-5",
  "author": "your-handle",
  "date": "YYYY-MM-DD",
  "score": 30,
  "witness": {
    "rows": ["01101...", "1011...", ...]
  }
}
```

- `rows[i]` 是长度为 n − 1 − i 的字符串，依次给出边 (i, i+1), (i, i+2), …, (i, n−1) 的颜色，即邻接矩阵的上三角部分；顶点从 0 开始编号，`'0'` 表示红色，`'1'` 表示蓝色。
- `rows` 共 n − 1 项（顶点 n−1 没有对应的行）。
- 任意 5 个顶点之间的 10 条边不得全部同色。

验证方式：`python3 verify.py <witness.json>`

## 文献纪录

R(5,5) ≥ 43 来自 Exoo 的 42 顶点染色（1989）。复现方法可参考 *Study of Exoo's Lower Bound for Ramsey number R(5,5)*（[arXiv:2212.12630](https://arxiv.org/abs/2212.12630)）。该 witness 尚未收录进 `records/`——提交一个有效的 42 顶点染色，即可拿下这里的已验证纪录。

## 已知方法

- 对边染色做局部搜索（以单色 K_5 的数量为目标函数的模拟退火或禁忌搜索），可以做到 35 个顶点上下。
- 最好的构造都是代数式的：先取 Paley 型图或分块循环染色，再做局部修补。Exoo 的 42 顶点 witness 正是基于循环构造。
- 快速评估每步改动至关重要：利用公共邻域位掩码技巧，维护每条边所参与的单色 K_5 计数。
